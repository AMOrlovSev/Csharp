Для подключения к БД SQLServer с логином и паролем в БД должен быть создано имя для входа с паролем.
Для подключения к базе Northwind этому имени задать соответствующую роль сервера (sysadmin).
В свойствах сервера на странице Безопасность указать Серверная проверка подлинности - Проверка подлинности SQL Server и Windows.
